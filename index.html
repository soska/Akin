Akin.js

Provides an easy to use and standard API to share links accross many social networks and services.

// share current page to Twitter
new Akin('twitter').share();

You can also pass some service specific options.

// define which user will be suggested by Twitter after sharing. And a default hashtag.
new Akin('twitter',{
        twitter : {
            related : ['soska', 'ioupon'],
            hashtags : ['ioupon']
        }
    }).share();

Services currently supported:

  • Twitter
  • Facebook (simple)
  • Email

Others are coming. Pull requests are accepted.
